# Weather App

A small app that tells you the weather for a zip code

![Weather Form](public/form.png)
![Weather Forecast](public/forecast.png)

## Setup

Install dependencies

```
./bin/setup
```

## Running

Run the app locally in development mode

```
./bin/dev
```

## Tests

Run the specs

```
bundle exec rspec
```
